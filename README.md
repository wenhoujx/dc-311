# What's this? 
this is a prototype of 311 map app using react and javascript 

# Build, Run, Deploy
The app is created using react-create-app, create see here for details on how to build, run and deploy it. 

[react-create-app](https://github.com/facebookincubator/create-react-app)

```
# commons 
yarn install 
yarn start 
# go to http:localhost:3000
```
